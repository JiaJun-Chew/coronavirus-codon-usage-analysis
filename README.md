# coronavirus-codon-usage-analysis
R scripts for codon usage bias analysis of coronavirus spike genes

# Codon Usage Bias Analysis in Coronaviruses

This repository contains R scripts used to analyze codon usage bias in coronavirus spike proteins.

## Analyses included:
- Kruskal-Wallis test 
- Correspondence Analysis
- Hierarchical clustering with pvclust
- ENC–GC3s neutrality analysis

## Requirements
- R (≥ 4.0)
- Packages: tidyverse, seqinr, pvclust, ggplot2, ggtree, FactoMineR

## Usage
Scripts can be run independently to reproduce figures and analyses presented in the manuscript.

## Citation
If you use this code, please cite the associated manuscript submitted to *PLOS ONE*.
